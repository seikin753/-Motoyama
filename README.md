# MuscleBoard (マッスルボード)

筋トレの成長を可視化し、タイムラインでの共有やフィードバックを通じてトレーニングを加速させるSNSアプリケーションです。
旧Google Apps Script (GAS) 版から、モダンな Next.js + Supabase スタックに完全に移行されました。

## 🚀 主な機能

### 📊 ダッシュボード
- **成長統計**: 連続トレーニング日数、今週の記録数、総記録数、ポイントの表示。
- **停滞検出**: 2週間以上PR（自己ベスト）更新がない種目を自動検出し、警告を表示。
- **本日の予定**: 今日記録したトレーニングのクイック確認。

### 💪 トレーニング記録
- **種目マスタ**: 部位別に整理された30種類以上のプリセット種目。
- **マルチセット入力**: セットごとの重量・回数を一括で記録。
- **前回コピー**: 前回のトレーニング内容をワンタップで反映。
- **体重記録**: トレーニング時の体重をあわせて記録可能。

### 📹 タイムライン (SNS)
- **メディア投稿**: 動画や画像をアップロードしてトレーニングを記録。
- **リアクション**: 🙏 (Helpful), ✨ (Want to try), 🔥 (Effective) の3種類のリアクション。
- **コメント**: 投稿に対するフィードバックや質問が可能。
- **ポイントシステム**: 投稿やアクションに応じてポイントが付与されます。

### 🏆 ランキング
- **成長率ランキング**: 前回比の重量増加率で競います。
- **ポイントランキング**: アプリ内活動の合計ポイント順。
- **自重比ランキング**: 体重に対する挙上重量の比率（例：ベンチプレス）を表示。

### 👤 プロフィール
- **ユーザー情報管理**: 名前、身長、体重、トレーニング歴の設定。
- **体重推移グラフ**: 過去の体重変化を視覚的に表示。
- **自己ベスト (PR)**: 全種目の最高記録と、Epley式による推定1RMの表示。

## 🛠 技術スタック

- **Frontend**: Next.js (App Router), TypeScript, Tailwind CSS
- **Backend/Database**: Supabase (PostgreSQL)
- **Auth**: Supabase Auth (Email/Password, Anonymous)
- **Storage**: Supabase Storage (動画・画像管理)
- **Deployment**: Vercel

## 📦 セットアップ

### 環境変数の設定
`.env.local` ファイルを作成し、以下の値を設定してください。

```env
NEXT_PUBLIC_SUPABASE_URL=あなたのプロジェクトURL
NEXT_PUBLIC_SUPABASE_ANON_KEY=あなたのANON_KEY
```

### インストールと起動
```bash
npm install
npm run dev
```

## 📄 ライセンス
